# Employee-management-system-mern-stack 

## Prerequisites
- Node
- npm/yarn

## How to start
Fork the project and download the code open your favorite IDE

1. Open terminal type command
```
cd server
```
``` 
npm i
```

2. Start the server

```
npm start
```

3. Split the terminal and type
```
cd client
```
``` 
npm i
```
```
npm start
```
